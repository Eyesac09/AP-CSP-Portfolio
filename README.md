# AP-CSP-Portfolio
A collection of beginner‑friendly to intermediate Python projects demonstrating logic, user interaction, file handling, randomness, and data analysis.
This repository includes multiple standalone programs, each designed to practice different programming concepts.

📌 Project List
1. 99 Bottles of Milk
A Python script that prints the full lyrics to “99 Bottles of Milk on the Wall.”  
Demonstrates loops, string formatting, and conditional grammar handling.

Features...
Counts down from 99 to 0
Handles plural vs. singular bottle wording
Prints the full song automatically

2. Dog Finder Program
A menu‑based program that helps users discover dog breeds based on:

Size preference
Breed group
Purpose (herding, guarding, hunting, etc.)
Uses a CSV dataset (dog_breed.csv) and pandas for filtering.

Key Concepts...
Data analysis with pandas
Random selection
User input menus
Filtering by weight, group, or purpose

3. Grocery Store Simulator
A small program that simulates grocery store operations.

Includes...
Price checking
Applying discounts
Filtering items by price
Calculating customer totals
Concepts Practiced
Lists
Indexing
Loops
Basic math operations

4. Superhero Nickname Generator
Interactive personality quiz that assigns a superhero identity based on user choices.

Demonstrates...
Nested conditionals
Branching logic
User input handling

5. Slot Machine Game
A fully interactive slot machine with:

Wager system
Free re‑roll mechanic
Jackpot and win conditions
Transaction history saved to a text file
Summary statistics (wins, losses, biggest win/loss, total profit)

Advanced Concepts
File I/O
Randomization
Game loops
Data persistence
Simulation mode (1000 auto spins)
